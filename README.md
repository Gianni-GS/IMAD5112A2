# IMAD5112A2

Background:

Users nowadays struggle to distinguish between productivity tips and urban myths that go viral. This app provides a “Life Hacks or Urban Myths”. This flashcard game helps users test their common knowledge about certain topics.

Objective:

The objective of this assignment was to create an Android App using Kotlin to help users to verify life hacks with flashcards. My app is a “Football Quiz App” that tests user’s knowledge on basic football questions. User must click “True or False” to the question on the top. After the 5 questions, there is a score board that will show users score out of 5. The user will also receive feedback on how well they did on the quiz.
